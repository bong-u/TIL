2631 : 줄세우기 (G4)

CREATED 2023-02-04

MODIFIED 2023-02-05

N = int(input())
L = []
for i in range(N):
    L.append(int(input()))

dp = [1]*N

for i in range(N):
    for j in range(i):
        if L[j] < L[i]:
            dp[i] = max(dp[i], dp[j]+1)

print (N-max(dp))
  • LIS를 활용한 문제이다
  • 풀이 유도는 잘했으나 LIS가 헷갈려서 전에 풀었던 코드를 참고하였다

### Design Techniques

#### Contextual Inquiry

- 사용자의 환경에서 사용자의 행동을 관찰

#### Design Funnel

- 아이디어를 확장함과 동시에 축소 시킴으로서 결과 도출

#### Double Diamond

- Discover -> Define -> Develop -> Deliver

#### Storyboarding

- 시나리오를 그림으로 표현

#### Prototyping

- 디자인을 표현하는 소프트웨어로 구현
- 종류: Low-fidelity(충실도가 낮음), High-fidelity(충실도가 높음)

#### User Testing

- In-lab vs On-site
- Moderated vs Unmoderated :
- Exploratory vs Assessment

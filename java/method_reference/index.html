메소드 참조

  • 메소드 참조는 Java 8부터 도입되었다
  • lambda와 같이 사용하면 간결한 코드를 만들 수 있다

정적 메소드 참조의 문법

1// lambda 식
2(str) -> String.toString(str);
3// 정적 메소드 참조
4String::toString

인스턴스 메소드 참조의 문법

1Person person;
2// lambda 식
3(age) -> person.setAge();
4// 인스턴스 메소드 참조
5person::setAge

활용

1// lambda식
2getItems.forEach(item -> System.out.println(item));
3// 메소드 참조
4getItems.forEach(System.out::println);
